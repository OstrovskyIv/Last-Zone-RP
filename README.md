### 🌟 Concept
A modern, atmospheric landing page for the **S.T.A.L.K.E.R.** RolePlay server. The project captures the gritty, industrial aesthetic of "The Zone" while maintaining high performance and responsiveness.

### 🛠 Key Features
*   **🃏 Interactive 3D Gallery:** Flip card effect using `preserve-3d` and `backface-visibility`. Cards rotate 180° on hover to reveal extra content.
*   **🛤 "How to Start" Section:** A step-by-step onboarding guide for players with adaptive arrows and thematic styling.
*   **🎨 Custom Typography:** Integration of specialized fonts like `capture-it` and `Caprasimo` to enhance the game's atmosphere.
*   **📱 Full Responsiveness:** Optimized for all screen sizes using Tailwind CSS grid and breakpoints.

### ⚠️ Performance Note
> **Please Note:** Currently, not all images have been converted to modern formats like `.webp`. Because of this, some high-resolution assets may take longer to load on slower connections.

### 💻 Tech Stack
*   **Framework:** Vue 3 (Composition API), TypeScript.
*   **Styles:** Tailwind CSS.
*   **Build Tool:** Vite.
*   **Deployment:** GitHub Actions.

### 🌟 Концепция
Современный и атмосферный лендинг для RolePlay сервера в сеттинге **S.T.A.L.K.E.R.** Проект передает суровую эстетику «Зоны», сохраняя при этом удобство навигации и адаптивность.

### 🛠 Ключевые фишки
*   **🃏 Интерактивная 3D-Галерея:** Секция с эффектом переворота карт (Flip Cards). Реализована через CSS `preserve-3d` для создания честного эффекта объема.
*   **🛤 Секция «Как начать играть»:** Пошаговая инструкция для игроков с адаптивными элементами, меняющими направление под разный размер экрана.
*   **🎨 Атмосферный дизайн:** Использование кастомных шрифтов (`capture-it`, `Caprasimo`) и графических эффектов в стиле игры.
*   **📱 Адаптивность:** Сайт полностью оптимизирован под мобильные устройства, планшеты и мониторы высокого разрешения.

### ⚠️ Оптимизация
> **Важно:** На данный момент не все изображения переведены в формат `.webp`. В связи с этим загрузка некоторых тяжелых ассетов может занять больше времени.
